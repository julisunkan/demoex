# Replit AI Agent Prompt вЂ“ Build MailVault Pro for Outlook

## Project Overview

Build a production-ready commercial Outlook Add-in named **MailVault Pro for Outlook**.

MailVault Pro is a Microsoft 365 SaaS application that enables Outlook users and organizations to securely:

- Backup Outlook emails
- Restore backed-up emails
- Perform selective backups
- Schedule automatic backups
- Analyze mailbox storage
- Mass delete emails safely
- Manage backup history
- Encrypt backups
- Support organizations with multiple users
- License the application using Microsoft Marketplace SaaS (AppSource)

The application must work with:

- Outlook Desktop
- Outlook on the Web
- Outlook for Microsoft 365

Do not use any third-party payment providers such as Stripe, PayPal, Paddle, LemonSqueezy, crypto, or similar. Subscription licensing must rely exclusively on Microsoft Marketplace SaaS APIs.
